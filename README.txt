USGA/GC Membership Dashboard

How to view locally:
1. Unzip this folder.
2. Open the folder named usga_gc_membership_dashboard_complete.
3. Double-click index.html.
4. Open it in Chrome, Edge, Safari, or Firefox.

Important:
- Keep the assets folder next to index.html.
- Do not move index.html out of the folder unless you also move the assets folder with it.
- The dashboard data and rendering logic are in assets/app.js.
- The styling is in assets/styles.css.

How to publish with GitHub Pages:
1. Create a new GitHub repository.
2. Upload index.html and the assets folder to the root of the repository.
3. Commit the files.
4. Go to Settings > Pages.
5. Choose "Deploy from a branch".
6. Select main and /root.
7. Save.
8. GitHub will provide the live dashboard URL after it builds.

If data does not appear:
- Make sure JavaScript is enabled.
- Make sure the assets folder is in the same directory as index.html.
- If you are using a corporate security environment, try Chrome or Edge first.
